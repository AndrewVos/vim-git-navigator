			for files in the current git repository.

==============================================================================

INTRODUCTION							*git-opener-introduction*

vim-git-opener is a command mode plugin that can fuzzy search files in your git
repository.

==============================================================================

COMMANDS							*git-opener-commands*

								*git-opener-:G*
:G {path}		Allows tab completion for files with any changes in the
			current git repository.
:Z {path}		Allows tab completion for any files in the current git
			repository.
								*git-opener-:Z*
